HTML

HTML (Hypertext Markup Language) is the standard markup Language fpr creating web pages and web applications.

HTML provides the structure and content of a webpage , defining elements such as:

  • Headings
  • Paragraphs
  • Links
  • Images
HTML5

HTML5 which is the latest version of HTML introduces new features and improvements over its predecessors. It provides tags and attributes that can define the various elements on web page, like headings,paragraphs, images, forms, links, and others.

HTML includes new elements such as

  • header
  • nav
  • article
  • section
  • footer

CSS

It is used to layout and style the visual presentation of HTML documents make it more beautiful.

It allows us to control the colors, fonts, spacing, and positioning of elements on a web page.

CSS can either be applied within an HTML document or externally in a separate stylesheet file.
